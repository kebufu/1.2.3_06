Run as java Executable file

Since this is a technical test, we didn't release this version to minecraft launcher, so you can't launch it through launcher, After downloading jar file and some necessary DLL, you can start it by double clicking jar file with 64 bit Java.If you install multiple JRE or openjdk, you need to configure environment variables to start it

Run as Command

After downloading jar file and some necessary DLL,You need to open a command prompt or a Linux terminal, CD to the directory where you store DLLs and jar files, and enter

java -jar 1.2.3_06.jar
